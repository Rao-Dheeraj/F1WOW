📧 EmailJS Setup - Alternative Methods

⚠️ Gmail Error: The Gmail API scopes error happens due to Google's security settings.
Solution: Use one of the easier options below!

🏆 OPTION 1: Custom SMTP (Recommended - Works with ANY email)

This works with any email provider - Gmail, Outlook, Yahoo, etc.

Step 1: Go to SMTPjs.com
Step 2: Click "Get Encryption Keys" (FREE - no signup needed)
Step 3: Enter your email credentials:
• Host: smtp.gmail.com (for Gmail)
• Username: Your full Gmail address
• Password: Use an App Password (not your regular password!)
🔑 How to create Gmail App Password:
1. Go to Google Account Security
2. Enable 2-Step Verification (if not enabled)
3. Search for "App Passwords"
4. Create one named "F1wow"
5. Copy the 16-character password
Step 4: SMTPjs will give you an Encryption Key - copy it!
Step 5: Paste the key below to test

📧 OPTION 2: EmailJS with Outlook (Easier than Gmail)

Use Outlook/Hotmail instead of Gmail - fewer permission issues!

Step 1: Create account at EmailJS.com
Step 2: Click "Add New Service" → Select Outlook or Mailgun
Step 3: Sign in with your Microsoft account
Step 4: Copy the Service ID
Step 5: Create template (same as before) and copy Template ID
Step 6: Get Public Key from Account settings

📬 OPTION 3: Mailgun (Professional - Free tier available)

More setup but very reliable - 5000 free emails/month!

Step 1: Sign up at Mailgun.com
Step 2: Verify your email domain
Step 3: Get your API Key from dashboard
This is the professional option - most reliable for production use!
